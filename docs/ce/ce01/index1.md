## 显示字符串“I Love Java” 
**【问题描述】** 
设计一个Java程序，显示一个字符串“I Love Java”   
**【输入形式】**
无  
**【输出形式】**
显示一行字符串“I Love Java”  
**【样例输入】**
无  
**【样例输出】**
无  
**【样例说明】**
无  
**【评分标准】**
字符串完全正确得20分  

```java
package love;

public class Love {
	public static void main(String[] args) {
		System.out.println("I Love Java");
	}
}
```
